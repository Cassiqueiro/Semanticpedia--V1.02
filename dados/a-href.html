Notas

Versões mais recentes de leitores de tela de 3 janelas:

  • JAWS
  • NVDA
  • Window Eyes

Versões mais recentes de 3 Navegadores:

  • Firefox
  • Chrome
  • Internet Explorer

Foram testadas 3 formas de interagir com os links:

  • Cursor: Usando as teclas up/down com as teclas da setas.
  • Tecla de Navegação: Usando as teclas de navegação do link dos leitores de tela ( U/V para JAWS e NVDA, U/L para olhar as janelas).
  • Tecla TAB: Usando o TAB do teclado.

Visão Geral dos Resultados

O anúncio do rótulo e da descrição dos links varia de acordo com o leitor de tela, o método de interação e o navegador.

  • Todos os leitores de tela anunciam o rótulo do link corretamente em todos os navegadores, independentemente do método de interação (exceto que o NVDA não anuncia o rótulo quando fornecido via aria-label (teste 2) quando o Chrome é usado).
  • O Window Eyes não anuncia a descrição, mas tem o anúncio de rótulo mais consistente em todos os métodos de interação e navegadores.
  • O JAWS só anuncia a descrição quando um usuário acessa um link.
  • O NVDA apenas anuncia a descrição ao usar as teclas de navegação de link ou teclas de tab no Firefox e Chrome. Ele não anuncia a descrição com o IE.
  • Os resultados nas tabelas que diferem da saída esperada são marcados.
