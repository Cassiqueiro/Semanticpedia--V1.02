Elementos Area | Imagem | Mapa

Material de apoio para desenvolvimento de jornadas

O elemento img e o atributo usemap apontam para o elemento map. Uma imagem. Elemento area


imagens (com alt="HTML5") com 2 links ('google' e 'Yahoo')



Código

<img src="http://www.html5accessibility.com/images/HTML5_Logo.png" usemap="#poot" alt="HTML5">
<map name="poot">
<area shape="rect" alt="google" coords="19,24,112,129" href="http://google.com">
<area shape="rect" alt="Yahoo" coords="19,4,114,24" href="http://search.yahoo.com/">
</map>
